Released scripts

ACRv2

2nd generation of Auto Coin Redeemer for Rocketbotroyale.

This application automatically collects your free 100 coin bonus in RBR every 30 min (real rates are around 30-35 min per collect). This can get you up to 4800 coins per day (This application averages around 4400 per day; A day here is 24 hours).

Made by thehermit

RBR Username Changer

A simple-to-use, fast, web application for changing your Rocketbotroyale display name in game.

This application changes your in-game display name in Rocketbotroyale, a feature not avalible in-game. You can also use this to add text decoration using special characters.


Made by Tank8k

Winterpixel Games Tools and Stats

A simple-to-use, fast, web application for viewing statistics for everything Winterpixel

This application uses real-time data from Rocketbotroyale, Gooberdash, Goobershot, Gooberroyale, and Moonrock miners, and shows them in an accesible, easy-to-use interface.

This website, while released and public, is still in-progress. As of 5/26/2024, only the Gooberdash tab has been finished.

Made by Tank8k

ACR

An Auto Coin Redeemer for Rocketbotroyale.

This application automatically collects your free 100 coin bonus in RBR every 30 min (real rates are around 30-31 min per collect). This can get you up to 4800 coins per day (This application averages around 4400 per day; A day here is 24 hours).

Made by thehermit

Custom Lobby Controller

An alternative to the RBR custom lobby UI.

This application acts like an alternative UI to the Rocketbotroyale custom lobby screen. It can control lobbies from another device, and has a few custom features that RBR does not have. For security reasons (Hormaaa), we are releasing this as an executable EXE file for windows 10/11. We will try to make releases for both Mac and Linux, along with a browser-accesible web version.

Projected release date: Before 6/10/2024 (depends)

Made by SHTT (shimobri, Hormaaa, thehermit, tuna)
